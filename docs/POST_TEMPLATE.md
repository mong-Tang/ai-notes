# POST_TEMPLATE

아래 템플릿으로 `posts_md/*.md`를 작성한다.  
기본 작성은 혼합형(한/영 병기 가능)으로 유지한다.

```md
note(요약): (필수) 핵심 한 줄

body(본문):
# 목표 (필수)
- 이번 글에서 확인할 것

# 실험 정보 (필수)
- 날짜:
- AI/모델:
- 도구/환경:

# 입력 (필수)
- 핵심 프롬프트/지시 요약:

# 성공 기준 (선택)
- 무엇을 성공으로 볼지:

# 결과 (필수)
- 출력 요약:
- 판정: 가능 / 조건부 가능 / 어려움

# 검증 (선택)
- 확인 방법:
- 검증 결과:
- 오류/환각 여부:

# 한계 (필수)
- 막힌 점 / 불안정 요소:

# 안전 체크 (선택)
- 민감정보 포함 여부:
- 출처/저작권 이슈 여부:

# 다음 액션 (필수)
- 다음에 바꿀 점 1~2개
```

작성 규칙
- 키워드는 `note(요약):`, `body(본문):` 권장
- 현재 파서는 `note:`, `요약:`, `note(요약):` / `body:`, `본문:`, `body(본문):` 모두 허용
- 필수 항목만 채워도 게시 가능, 선택 항목은 필요 시 확장
