요약:
새글을 추가하는방법
본문:
PS D:\my_Work\workspace\use_AI> python scripts/new_post.py posts/crt-post-md

PS D:\my_Work\workspace\use_AI> = 작업경로
python scripts/new_post.py posts/ = 파이썬 명령(ports/ = 폴더)
crt-post-md = 파일명(.md)
*파일명에 .md 를 생략해도 .md가 붙는다. 본래는 .md를 붙여야함.

내용과 본문을 작성후 파이썬을 구동한다
python scripts/generate_post.py posts_md/새글이름.md
